# 止损止盈与仓位管理规则

## 核心原则

> **每笔交易的最大亏损不超过总资金的2%（严格执行）**
> 风险收益比至少1:2，理想1:3以上

---

## 止损位设置方法

### 方法1：技术止损（首选）

根据关键技术位设置，止损位放在该位置下方一定缓冲：

**支撑位止损**
```
止损位 = 关键支撑位 × (1 - 缓冲系数)
缓冲系数 = 1-2%（流动性好的大盘股用1%，小盘股用2%）
```

**均线止损**
```
跌破MA20收盘价 → 第一止损信号
跌破MA60收盘价 → 强制止损（无论亏损多少）
```

**前低止损**
```
止损位 = 最近一个重要低点 - 缓冲（约0.5-1%）
```

### 方法2：波动率止损（ATR止损）

```
ATR14（14日平均真实波幅）
止损位 = 入场价 - 2×ATR14
```

适用场景：趋势行情，支撑位不明显时

### 方法3：百分比止损（最后防线）

```
固定止损比例 = -5%（从入场价计算）
任何情况下跌超过5%必须止损，不等待
```

## 止盈策略

### 目标位设置方法

**方法1：关键压力位止盈**
```
第一目标 = 最近一个重要压力位（前高、密集成交区上沿）
第二目标 = 更长周期压力位（近半年高点、年线压力等）
```

**方法2：风险收益比推算**
```
已知：入场价、止损位
止损空间 = 入场价 - 止损位
第一目标 = 入场价 + 止损空间 × 2（1:2）
第二目标 = 入场价 + 止损空间 × 3（1:3）
```

**方法3：均线扩张目标**
```
在强趋势中，价格往往在布林带上轨附近遇阻
上轨 = MA20 + 2×标准差（动态计算）
```

### 移动止盈规则

**保本止损升级**
```
当浮盈达到入场价的+5%时，止损上移至入场价（保本）
当浮盈达到+10%时，止损上移至入场价+3%（锁定部分利润）
```

**趋势跟踪止盈**
```
在趋势行情中，将止损持续上移到最近一个低点下方
收益达到第一目标时，减仓50%，剩余仓位用移动止盈跟踪
```

## 仓位管理框架

### 基础仓位计算公式

```
允许亏损金额 = 总资金 × 2%（固定风险）
每股亏损空间 = 入场价 - 止损价
理论仓位（股数）= 允许亏损金额 ÷ 每股亏损空间
初始仓位（金额）= 理论仓位 × 入场价
初始仓位（占总资金%）= 初始仓位金额 ÷ 总资金
```

**示例：**
```
总资金 = 100万元
入场价 = 20.00元
止损价 = 19.00元（-5%）
每股亏损空间 = 1.00元

允许亏损金额 = 100万 × 2% = 2万元
理论仓位 = 2万 ÷ 1.00 = 2万股
初始仓位金额 = 2万股 × 20元 = 40万元
初始仓位比例 = 40% （占总资金）
```

### 分档建仓策略（三段式）

避免一次性满仓，降低建仓成本波动：

| 建仓阶段 | 占总仓位 | 触发条件 |
|---------|---------|---------|
| 第一批 | 30-40% | 满足入场条件，信号出现 |
| 第二批 | 30-40% | 价格上涨确认，突破关键位 |
| 第三批 | 20-30% | 趋势加速，回踩不破 |

**加仓原则：**
- 只在盈利状态下加仓（浮盈+3%以上）
- 加仓价格必须高于前一批入场价
- 加仓后重新计算整体止损位（不能倒退）
- 不允许在亏损状态下摊平成本（A股中非常危险）

### 综合评分与仓位对照

| 综合评分 | 建议初始仓位 | 最大仓位 | 操作属性 |
|---------|------------|---------|---------|
| 85-100 | 30-40% | 70% | 积极做多 |
| 70-84 | 20-30% | 50% | 谨慎做多 |
| 55-69 | 10-20% | 30% | 试探性建仓 |
| 40-54 | 0% | 10% | 观望为主 |
| < 40 | 0% | 0% | 严格观望 |

## 混合风格的周期切换规则

由于用户采用混合风格（根据机会切换），以下规则帮助判断当前信号属于哪个周期：

### 短线信号特征（持仓1-5天）
- 触发条件：日线出现明确K线形态信号（如早晨之星、锤子线等）
- 使用指标：KDJ低位金叉、MACD柱状图从绿转红
- 仓位控制：最高30%
- 止损：更紧，-3%以内
- 必备条件：日内量能配合，换手率 > 3%

### 波段信号特征（持仓2-6周）
- 触发条件：日线MACD金叉 + 周线趋势方向一致
- 使用指标：MA20支撑 + 布林带开口 + RSI突破50
- 仓位控制：30-50%
- 止损：中等，跌破MA20收盘价
- 必备条件：周线MACD在零轴上方或向上穿越

### 中长线信号特征（持仓1-6个月）
- 触发条件：月线级别底部形态 + 周线金叉 + 日线突破
- 使用指标：年线支撑 + 底部放量 + 三周期共振
- 仓位控制：30-60%，可分3批建仓
- 止损：宽松，跌破MA60收盘价
- 必备条件：基本面支撑（结合铝行业分析agent）
