# 技术指标详细解读规则

## MACD（Moving Average Convergence Divergence）

### 参数设置
- 标准参数：EMA12, EMA26, Signal9
- 混合风格额外参考：日线MACD + 周线MACD双重确认

### 核心解读逻辑

**零轴位置（最重要）**
- DIF和DEA均在零轴上方 → 处于多头市场，做多优先
- DIF和DEA均在零轴下方 → 处于空头市场，做多需谨慎
- DIF穿越零轴向上 → 中期趋势由空转多，较强信号

**金叉死叉**
- 零轴上方金叉 → 强烈多头信号（优先级最高）
- 零轴下方金叉 → 反弹信号，非趋势性（降权）
- 零轴上方死叉 → 回调信号，不一定反转
- 零轴下方死叉 → 继续下跌，避免抄底

**MACD柱状图（Histogram）**
- 红柱由短变长（扩张）→ 上涨动能增强
- 红柱由长变短（收缩）→ 上涨动能减弱，注意减仓
- 绿柱由长变短 → 下跌动能减弱，可能企稳
- 绿柱转红柱 → 金叉确认

**背离信号（高可信度反转信号）**
- 顶背离：价格创新高，MACD未创新高 → 上涨乏力，警惕顶部
- 底背离：价格创新低，MACD未创新低 → 下跌动能衰竭，可能反转
- 背离出现后需等待价格确认（如跌破均线或突破压力）才操作

## KDJ

### 参数设置
- 标准参数：K9, D3, J3（A股默认）

### 核心解读逻辑

**超买超卖区间**
- K值和D值均 > 80：超买区，短期可能回调（但强趋势中可长期处于超买）
- K值和D值均 < 20：超卖区，短期可能反弹
- 注意：强趋势中超买超卖信号可靠性下降，需结合趋势判断

**金叉死叉**
- 20以下低位金叉 → 可靠买入信号
- 50附近金叉 → 趋势延续确认
- 80以上高位死叉 → 短期见顶信号
- J值大于100或小于0 → 极端超买/超卖，反转概率提高

**背离（与MACD背离逻辑相同，但更灵敏）**

## RSI（Relative Strength Index）

### 参数设置
- 主要参考：RSI14（日线）
- 辅助参考：RSI6（更灵敏）、RSI21（更平滑）

### 核心解读逻辑

**关键区间**
- RSI > 70：超买，注意风险
- RSI 50-70：多头区间，趋势健康
- RSI = 50：多空分界线，突破50向上是积极信号
- RSI 30-50：空头区间，反弹谨慎参与
- RSI < 30：超卖，可能反弹但不代表趋势反转

**50轴的重要性**
- RSI从下方突破50轴 → 中期转多信号
- RSI跌破50轴 → 中期转空信号
- RSI在50附近来回 → 趋势不明，观望

**背离（与MACD背离类似，RSI背离更常见但假信号更多）**

## 布林带（Bollinger Bands）

### 参数设置
- 标准参数：MA20, 2倍标准差

### 核心解读逻辑

**带宽变化**
- 布林带开口扩大（带宽增加）→ 趋势行情启动/延续
- 布林带收口收窄（带宽减小）→ 盘整，即将变盘
- 带宽极度收窄（Squeeze）→ 大行情前兆，方向待定

**价格与轨道关系**
- 价格沿上轨运行（上轨贴合）→ 强势上涨趋势，不宜做空
- 价格突破上轨后回落至中轨 → 正常回调，可加仓
- 价格从上轨跌破中轨 → 趋势减弱信号
- 价格在下轨附近反复测试不破 → 支撑有效，可能反弹
- 价格沿下轨运行 → 强势下跌，不宜做多

**中轨（MA20）**
- 在上升趋势中，MA20是最重要的短期支撑
- 价格回踩MA20不破并再次上涨 → 趋势延续，买点

## 均线系统

### 关键均线及其意义

| 均线 | 别称 | 主要意义 |
|------|------|---------|
| MA5 | 五日线 | 超短期趋势，日内和T+1参考 |
| MA10 | 十日线 | 短期趋势，波段操作参考 |
| MA20 | 月线/布林中轨 | 中短期趋势，最重要的操作均线 |
| MA60 | 季线 | 中期趋势分界线 |
| MA120 | 半年线 | 中长期趋势分界 |
| MA250 | 年线 | 牛熊分界线，强支撑/压力 |

### 均线形态解读

**多头排列（MA5 > MA10 > MA20 > MA60）**
- 表明各周期趋势方向一致向上
- 回调时均线提供支撑，是买点
- 多头排列+放量上涨 → 最强烈的做多信号

**空头排列（MA5 < MA10 < MA20 < MA60）**
- 各周期趋势一致向下
- 反弹时均线提供压力
- 空头排列中的反弹建议卖出，不建议追多

**均线粘合（多条均线集中在一个价格区间）**
- 蓄势待发，即将突破
- 突破方向决定后续趋势

**均线金叉死叉（常用：MA5/MA20, MA10/MA60）**
- 短期线从下方穿越长期线向上 → 金叉，多头信号
- 短期线从上方穿越长期线向下 → 死叉，空头信号
- 黄金交叉（MA50上穿MA200）是非常强的长期信号

## 成交量分析

### 量价关系四种基本模式

| 价格 | 成交量 | 含义 | 操作 |
|------|--------|------|------|
| 上涨 | 放量 | 量价齐升，趋势健康 | 持有/追多 |
| 上涨 | 缩量 | 上涨动能不足 | 谨慎，注意止盈 |
| 下跌 | 放量 | 主动抛压大，空头强势 | 止损/观望 |
| 下跌 | 缩量 | 惜售，抛压减轻 | 可能企稳，密切观察 |

### 特殊量能信号

**地量**（成交量萎缩至近期极低水平）
- 在底部区域出现 → 筹码出清，卖方枯竭，反转信号
- 在上升趋势中的回调 → 缩量回调健康，可加仓

**天量**（成交量放大至近期极高水平，换手率异常）
- 在高位出现 + 价格涨幅缩小 → 主力出货信号，警惕顶部
- 在底部出现 + 价格大幅上涨 → 主力建仓，启动信号

**量堆（多日成交量维持高位）**
- 上涨过程中量堆 → 趋势确立，可持有
- 高位量堆后价格滞涨 → 可能是顶部震荡出货

### 换手率参考标准（A股）
- 日换手率 < 1%：成交清淡
- 日换手率 1-3%：正常水平
- 日换手率 3-7%：活跃
- 日换手率 > 7%：高度活跃，需结合价格判断
- 日换手率 > 15%：异常，警惕主力操作
