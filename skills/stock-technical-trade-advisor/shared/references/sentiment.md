# 市场情绪指标解读规则

## 北向资金（沪深港通北向）

### 数据获取
- 搜索词：`[股票名称] 北向资金 持股 陆股通`
- 可靠来源：东方财富、同花顺、Wind（付费）
- 关注维度：单日净买入额、5日累计、持股比例变化

### 解读逻辑

**单日净买入/卖出**
- 净买入 > 5000万元：明显增仓，正面信号
- 净买入 1000-5000万：温和增仓
- 0附近波动：中性
- 净卖出 1000-5000万：温和减仓
- 净卖出 > 5000万：明显减仓，负面信号

**5日累计趋势**
- 连续5日净买入：外资持续增仓，趋势性信号，权重高
- 连续5日净卖出：外资持续撤离，需警惕

**持股比例变化**
- 持股比例从历史低位回升：外资重新看好，参考价值高
- 持股比例从历史高位下降：外资获利了结或转向

### 局限性
- 北向资金有时受汇率波动、全球风险情绪影响，并非完全因为标的本身
- 北向资金在中小盘股中的参考价值高于大盘蓝筹（大盘股北向持仓大，轻微变化不代表态度）
- 不能将北向资金作为唯一判断依据

## 融资数据（两融中的融资买入）

### 数据获取
- 搜索词：`[股票名称] 融资余额 最新`
- 来源：交易所官网每日公告、东方财富融资融券数据

### 核心指标

**融资余额**
```
融资余额 = 所有融资买入尚未偿还的金额
融资余额增加 → 场内杠杆资金在买入看多
融资余额减少 → 去杠杆，多头情绪降温
```

**融资余额占流通市值比（融资比例）**
```
融资比例 = 融资余额 / 流通市值
< 1%：杠杆使用低，主力未进场
1-3%：正常区间
3-5%：多头热情较高，需关注
> 5%：高杠杆，追高风险大，一旦下跌可能引发强平踩踏
> 8%：极度危险，历史上多次成为阶段性顶部信号
```

**融资买入额（当日）**
- 单日融资买入额大幅放量（超过近5日均值2倍以上）→ 短期看多情绪高涨，但也可能是顶部特征

### 融资数据的反向思考
- 在底部区域，融资余额往往因为套牢盘不断止损而降至极低 → 反而是筹码出清的信号
- 在顶部区域，融资余额往往高企 → 反映散户追高
- **最危险的状态**：股价在高位 + 融资余额创新高 + 北向资金在减仓

## 龙虎榜数据

### 触发龙虎榜的条件（A股规则）
- 涨幅超过7%（非ST股票）或5%（ST股票）
- 跌幅超过7%或5%
- 换手率超过20%
- 连续三个交易日涨跌幅超过5%

### 席位解读方法

**机构席位**
- 显示为「机构专用」
- 多个机构席位同时买入 → 强烈多头信号，往往预示后续上涨
- 机构席位大量卖出（尤其在高位）→ 出货信号

**知名游资**
- 常见知名游资席位：华鑫证券上海分公司、中信证券上海分公司等
- 知名游资集体进场 → 短期强势信号（但往往是快进快出）
- 跟随游资需注意：他们的持仓周期往往只有1-3天

**营业部统计**
- 买入前5席位净买入 vs 卖出前5席位净卖出
- 整体净流入（买入>卖出）→ 多头主导
- 整体净流出（卖出>买入）→ 空头主导

### 龙虎榜后续走势统计规律（A股历史经验）
- 机构大量买入后次日：上涨概率约65%
- 游资扎堆买入后次日：上涨概率约55%，但波动大
- 机构大量卖出后次日：下跌概率约60%
- 涨停次日有龙虎榜且机构净买入：较大概率继续上涨1-3日

## 主力资金流向

### 数据定义（各平台定义略有差异）
- 超大单（主力）：单笔 > 100万元
- 大单：单笔 50-100万元
- 中单：单笔 10-50万元
- 小单（散户）：单笔 < 10万元

### 解读规则

**主力资金净流入**
```
主力净流入 = 超大单+大单净买入额
连续3日主力净流入 + 价格上涨 → 主力推升，趋势延续
主力净流入 + 价格不涨（滞涨）→ 可能是对倒或洗盘
```

**散户与主力背离**
- 主力净流入 + 散户净流出 → 主力建仓，散户出逃（往往是买点）
- 主力净流出 + 散户净流入 → 主力出货，散户接盘（往往是卖点）

### 注意事项
- 主力资金流向数据来自大单拆分推算，并非100%准确
- 不同数据平台（东财、同花顺）计算口径不同，数值可能有差异
- 参考价值：短期信号参考 > 中长期判断依据

## 情绪综合评分计算

| 维度 | 权重 | 评分逻辑 |
|------|------|---------|
| 北向资金（5日趋势） | 35% | 持续净买入100分→持续净卖出0分 |
| 融资余额变化 | 25% | 适度增加高分，过高或减少低分 |
| 龙虎榜席位性质 | 25% | 机构净买入100分，机构净卖出0分 |
| 主力资金流向 | 15% | 净流入高分，净流出低分 |

**评分注意：融资余额不是越高越好**
- 融资余额增加且融资比例 < 3%：加分
- 融资余额增加且融资比例 3-5%：中性
- 融资余额增加且融资比例 > 5%：减分（风险加剧）

## A股特有情绪信号

### 涨停板分析
- 一字涨停（开盘即封板）：强势，筹码锁定，后续上涨概率高
- 开盘涨停后打开再封回：分歧后强势，关注换手率
- 涨停打开未封回：弱势涨停，次日高开低走概率大

### 跌停板分析
- 一字跌停：恐慌情绪极端，但可能也是底部清洗
- 跌停打开：卖压减轻，可能阶段性企稳

### 连板效应（A股特色）
- 连续涨停板数：2板以上开始产生赚钱效应，资金追捧
- 龙头股效应：板块内连板数最多的个股，资金聚集效应最强
- 注意：参与连板股属于短线高风险操作，非本agent的主要分析场景
