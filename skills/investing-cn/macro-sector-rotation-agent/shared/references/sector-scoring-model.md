# 板块评分模型

## 一、评分维度与权重

| 维度 | 权重 | 说明 |
|---|---|---|
| 宏观匹配度 | 25% | 板块与宏观四象限、流动性状态的适配 |
| 轮动强度 | 25% | 趋势动量和相对强弱 |
| 基本面改善 | 20% | 景气与盈利预期边际 |
| 估值性价比 | 15% | 当前估值分位与历史比较 |
| 政策/事件催化 | 15% | 催化确定性、持续性、可交易性 |

总分：
```
sector_score = macro*0.25 + rotation*0.25 + fundamentals*0.20 + valuation*0.15 + catalyst*0.15
```

---

## 二、宏观匹配度规则（0-100）

### 复苏（增长上行，通胀温和）
- 偏好：顺周期（有色、化工、机械）、可选消费、科技制造
- 防御：公用事业、高股息相对中性

### 过热（增长上行，通胀抬升）
- 偏好：资源、能源、上游定价权板块
- 回避：高久期高估值成长

### 滞胀（增长下行，通胀上行）
- 偏好：能源、公用事业、红利防御
- 回避：高成本、弱现金流行业

### 衰退（增长下行，通胀下行）
- 偏好：利率敏感成长（若流动性改善）、高分红防御
- 回避：高杠杆强周期

---

## 三、评级阈值与配置建议

| 总分 | 评级 | 建议 |
|---|---|---|
| >= 80 | 强超配 | 主线配置，优先资金投入 |
| 70-79 | 超配 | 重点配置 |
| 60-69 | 标配 | 维持中性仓位 |
| 50-59 | 低配 | 仅交易性参与 |
| < 50 | 回避 | 原则上不配置 |

---

## 四、风险校正

当出现以下情况，对最终评分做扣分：
- 估值拥挤明显：-5 到 -12
- 政策兑现弱于预期：-5 到 -10
- 国际风险快速恶化：-5 到 -15
- 成交额不足导致趋势脆弱：-3 到 -8

最终分数：
```
final_score = sector_score - risk_penalty
```

---

## 五、输出模板

```
板块: [名称]
初始得分: [0-100]
风险扣分: [-x]
最终得分: [0-100]
评级: 强超配/超配/标配/低配/回避
主驱动: [1-2条]
失效条件: [1-2条]
执行周期: 短期(1-4周)/中期(1-3个月)
置信度: 高/中/低
```

## 六、仓位表达规范（避免过度精确）

若用户要求仓位建议，使用区间表达：
- 强超配：相对基准 +5% 到 +10%
- 超配：相对基准 +2% 到 +5%
- 标配：相对基准 -2% 到 +2%
- 低配：相对基准 -5% 到 -2%
- 回避：相对基准 -10% 到 -5%

若用户未提供风险偏好，不输出绝对仓位，仅输出“相对基准仓位”。
