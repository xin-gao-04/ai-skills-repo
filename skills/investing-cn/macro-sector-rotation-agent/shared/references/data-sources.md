# 宏观板块轮动数据源清单

## 一、宏观与政策数据

### 国家统计局（首选）
- 数据：PMI、CPI、PPI、工业增加值、社零
- 搜索触达：`国家统计局 PMI 最新`、`国家统计局 CPI PPI 最新`
- 频率：月度

### 中国人民银行 + 国家金融监管口径
- 数据：社融存量、M1/M2、LPR、货币政策表态
- 搜索触达：`社融 存量增速 最新`、`LPR 最新`
- 频率：月度/不定期

### 国务院/部委政策发布
- 数据：产业政策、财政刺激、监管政策
- 搜索触达：`国务院 政策发布 最新`、`工信部 产业政策 最新`
- 意义：政策催化主线识别

## 二、全球流动性与海外变量

### 美联储与美国宏观

## 六、数据采集优先级

```
高频（每次分析必查）:
  宏观核心变量 + 资金面 + 风格指数 + 申万行业涨跌

中频（每周更新）:
  行业景气高频 + 政策与地缘事件梳理

低频（每月更新）:
  宏观月度数据、盈利预测系统性修正、估值分位刷新
```

## 七、来源可靠性分层

- `A层（官方/交易所/权威机构）`：国家统计局、人民银行、交易所、央行/Fed官网
- `B层（主流金融终端/媒体）`：券商终端、主流财经媒体、行业协会
- `C层（二次转载/自媒体）`：仅用于线索，不可单独作为结论依据

结论使用规则：
- 关键结论至少需要 1 个 A层 或 2 个 B层 互相验证
- 若仅有 C层来源，必须标注“未证实”

## 八、交易日与时区规则

- A股交易日优先使用北京时间（UTC+8）
- 周末/节假日无日频行情时，使用最近交易日收盘数据
- 跨市场变量（美债、DXY、油价）需注明当地时间与北京时间对应关系
