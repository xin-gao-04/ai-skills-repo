# 铝行业投资分析 Agent

## 工作流概述

本 Agent 分为 **5个串行阶段**，每阶段产出结构化数据，最终汇总为投资决策报告。

```
阶段1: 价格与库存数据采集
    ↓
阶段2: 成本-利润动态测算
    ↓  
阶段3: 周期位置判断
    ↓
阶段4: 标的比较与估值
    ↓
阶段5: 生成投资报告
```

详细的数据源清单见 `references/data-sources.md`
利润弹性计算逻辑见 `references/profit-model.md`
标的评分规则见 `references/scoring-model.md`

---

## 阶段1：价格与库存数据采集

**目标：** 获取驱动铝行业盈利的全部关键变量的当前值。

### 1.1 价格数据（优先级最高）

依次用 web_search 获取以下数据，搜索词见括号：

| 数据项 | 搜索词 | 期望获取 |
|--------|--------|---------|
| 沪铝主力价格 | `沪铝主力合约今日收盘价` | 元/吨，当日 |
| LME铝价 | `LME aluminum price today` | 美元/吨，当日 |
| 氧化铝现货价 | `氧化铝现货价格 今日 SMM` | 元/吨，当日 |
| 内外价差 | 根据沪铝与LME价格换算（乘汇率） | 元/吨差值 |

如果搜索结果含有上海有色网(smm.cn)、世铝网(cnal.com)或长江有色的数据，优先采用。

### 1.2 库存数据（供需信号）

| 数据项 | 搜索词 | 期望获取 |
|--------|--------|---------|
| 铝锭社会库存 | `SMM 铝锭社会库存 本周` | 万吨，与上周变化 |
| 铝棒库存 | `SMM 铝棒库存 本周` | 万吨 |
| LME铝库存 | `LME aluminum inventory` | 吨，近期变化 |
| 铝锭现货升贴水 | `长江铝 现货升贴水` | 元/吨，正为升水负为贴水 |

**库存判断逻辑：**
- 铝锭库存 < 50万吨且持续去库 → 供需偏紧，利多价格
- 铝锭库存 > 80万吨或累库加速 → 短期压制价格  
- 现货升水 > 0 → 即期需求强，多头信号
- 现货贴水 > 150元/吨 → 需求疲软，注意风险

### 1.3 成本端数据

| 数据项 | 搜索词 | 用途 |
|--------|--------|------|
| 动力煤价格 | `秦皇岛动力煤价格 今日` | 影响火电铝成本 |
| 云南用电政策 | `云南电解铝 开工率 限电` | 影响供给量 |
| 电解铝行业平均成本 | `安泰科 电解铝完全成本 最新` | 成本支撑判断 |

### 1.4 宏观与政策数据

| 数据项 | 搜索词 |
|--------|--------|
| 人民币/美元汇率 | `USD CNY 汇率 今日` |
| 美联储最新表态 | `Federal Reserve interest rate latest` |
| 国内制造业PMI | `中国制造业PMI 最新` |
| 铝行业政策 | `电解铝 产能 政策 最新` |

**采集完成后，将以上数据填入 `data_snapshot` 字典，传入阶段2。**

## 阶段2：成本-利润动态测算

**目标：** 根据当前价格与成本数据，动态测算各主要标的的实时盈利水平和弹性空间。

### 2.1 行业利润测算

```
电解铝吨利润 = 铝现货价（不含税）- 综合成本（含税→除以1.13换算）

综合成本 = 氧化铝成本 + 电力成本 + 阳极成本 + 其他
        = 氧化铝价×1.93 + 电价×13500 + 阳极价×0.47 + 700（其他）
```

如果无法获得精确分项数据，使用行业加权平均完全成本作为代理（安泰科月度数据）。

**行业利润区间判断：**
- 吨利润 < 500元 → 行业亏损边缘，供给有减产风险 → **潜在供给收缩利多**
- 吨利润 500-2000元 → 正常盈利，行情稳定
- 吨利润 2000-4000元 → 高盈利，刺激复产扩产 → **注意供给压力**
- 吨利润 > 4000元 → 超高利润，警惕利润回归风险

### 2.2 各标的盈利弹性测算

读取 `references/profit-model.md` 获取各标的的基础参数（产量、成本结构），
然后代入当前价格计算各公司实时利润和对价格变动的敏感性。

**输出格式：**
```
标的: 中国铝业 (601600)
当前铝价假设: X 元/吨
当前氧化铝价: Y 元/吨
估算吨利润: Z 元/吨
估算年化归母净利润: W 亿元
铝价+500元/吨 → 净利润增量: +A 亿元
铝价-500元/吨 → 净利润减少: -B 亿元
```

对4个核心标的分别输出以上内容。

## 阶段3：周期位置判断

**目标：** 综合多个维度，判断当前铝市处于哪个周期阶段，以及操作窗口属性。

### 3.1 周期判断框架（四象限）

```
                 基本面偏多（去库/供紧）
                        |
  价格在成本支撑附近 ←——+——→ 价格在历史高位
  （低位多头区间）        |    （高位观望区间）
                        |
                 基本面偏空（累库/供松）
```

根据阶段1采集的数据，输出：
- **基本面方向：** 偏多 / 中性 / 偏空（依据库存变化、利润水平）
- **价格位置：** 低位 / 中位 / 高位（与近3年均价和成本线对比）
- **宏观方向：** 顺风（美元弱、国内宽松） / 逆风
- **季节性因素：** 当前月份对应的历史旺季/淡季规律

### 3.2 周期阶段输出

基于以上判断，给出一个简洁的周期定性：

**示例输出：**
> 当前判断：**价格高位 + 基本面中性偏空 + 宏观中性**
> 周期阶段：价格领先基本面，金融属性主导，产业面暂不支撑当前价位。
> 操作属性：短期震荡消化，等待基本面修复或回调至成本+合理利润区间。

## 阶段4：标的比较与估值

**目标：** 横向比较4个核心铝标的，给出相对吸引力排名和估值判断。

### 4.1 标的基础档案

从 `references/scoring-model.md` 载入各标的基础档案（成本、产能、估值历史）。

### 4.2 当前估值计算

搜索各标的最新股价和最新业绩数据：
- 搜索词：`中国铝业 601600 今日股价` / `云铝股份 000807 今日股价` 等

计算动态PE：
```
动态PE = 当前股价 × 总股本 / 估算年化净利润（来自阶段2）
```

### 4.3 综合评分（满分100分）

参考 `references/scoring-model.md` 中的权重，对每个标的打分：

| 维度 | 权重 | 评分依据 |
|------|------|---------|
| 成本竞争力 | 25% | 吨成本在行业分位数 |
| 利润弹性 | 20% | 铝价+1000元/吨对应净利弹性% |
| 当前估值 | 25% | 动态PE与历史均值比较 |
| 基本面匹配度 | 15% | 当前周期阶段与标的特性匹配 |
| 风险因子 | 15% | 电力依赖、海外风险、流动性 |

### 4.4 标的排名输出

输出标的吸引力排名，附简要理由。

## 阶段5：生成投资报告

**目标：** 整合前4个阶段输出，生成结构化的投资决策报告。

### 5.1 报告结构（ALWAYS 使用此模板）

```markdown
# 铝行业投资周报
**报告日期：** YYYY-MM-DD
**报告类型：** 定期跟踪 / 触发预警

---

## 一、本期关键变量速览
[用表格呈现核心数据：价格、库存、成本、较上期变化]

## 二、行业基本面研判
[2-3段文字，覆盖：供给端变化、需求端变化、库存周期判断]

## 三、利润弹性测算
[各标的当前盈利水平，以及铝价±500/1000元/吨的情景分析]

## 四、周期位置与操作窗口
[明确当前所处阶段，以及未来1-4周的主要关注点和触发条件]

## 五、标的比较与推荐
[标的评分表 + 相对吸引力排名 + 重点标的操作建议]

## 六、风险提示
[列出3-5个需要跟踪的尾部风险]

## 七、下期关注重点
[明确下次分析前需要跟踪的关键数据和事件]
```

### 5.2 操作建议格式

操作建议必须包含以下要素，不得含糊：

```
标的：[名称+代码]
建议操作：买入 / 持有 / 减仓 / 观望
时间维度：短期（1-4周）/ 中期（1-3个月）/ 长期（6-12个月）
核心逻辑：[1-2句话，说明为什么]
关键条件：触发此操作建议成立的前提条件
止损/止盈参考：[具体价格区间或估值参数，仅供参考]
风险声明：本建议仅为分析工具输出，不构成投资建议，请结合自身情况判断。
```

### 5.3 报告输出

将报告保存为 Markdown 文件，文件名格式：
`aluminum-weekly-YYYY-MM-DD.md`

同时在对话中给出报告摘要（核心结论，不超过300字）。

如果用户希望将此分析流程设置为定时任务，在报告末尾附上以下指引：

> **定时任务配置建议：**
> 
> **推荐频率：** 每周一次（周五收盘后或周一开市前）
> 
> **触发方式（Claude.ai）：** 
> 可将本 SKILL.md 作为 System Prompt 或 Project 知识库，
> 届时只需发送 `执行本周铝市分析` 即可触发完整工作流。
> 
> **触发方式（API调用）：**
> 参考附录 `references/api-trigger.md` 中的 Python 脚本，
> 可配合 cron 实现完全自动化。
> 
> **预警触发条件（随时触发，不等周期）：**
> - 铝价单日涨跌超过 3%
> - 铝锭社会库存周变化超过 ±10万吨
> - 氧化铝价格单周涨跌超过 5%
> - 云南出现限电减产公告
> - 美联储出现超预期政策变化

## 执行注意事项

1. **数据时效性：** 优先获取当日或本周数据，明确标注数据日期
2. **无法获取的数据：** 若某项数据搜索失败，用最近一次已知值 + 说明代替，不要跳过
3. **不确定性处理：** 对于判断性结论（如周期位置），明确说明支撑依据和反驳依据
4. **免责声明：** 报告末尾必须包含投资风险声明
5. **中文输出：** 全部报告内容使用中文，专业术语保留英文缩写（如 LME、PE）
