# 铝行业数据源清单

## 一、核心价格数据源

### 上海有色网 SMM（首选）
- 网址：https://hq.smm.cn/h5/alu
- 数据：铝锭现货价、沪铝期货、LME铝价、铝棒、铝合金价格
- 更新：每交易日实时更新
- 搜索触达：`SMM 铝价 今日` / `上海有色 铝锭价格`

### 世铝网 cnal.com（次选）
- 网址：https://market.cnal.com/
- 数据：多市场铝价汇总（长江、南海、上海、广东）、LME实时价格
- 更新：每交易日
- 搜索触达：`世铝网 铝价 今日` / `cnal.com 铝锭`

### 长江有色金属网
- 网址：https://alu.ccmn.cn
- 数据：长江现货A00铝锭报价，历史最悠久的现货基准
- 搜索触达：`长江现货铝价` / `长江A00铝锭`

---

## 二、库存数据源

### SMM铝锭社会库存（最权威）
- 发布时间：每周四下午发布上周四数据
- 搜索触达：`SMM 铝锭社会库存 本周` / `电解铝锭库存 最新`
- 覆盖地区：无锡、上海、杭州、南海、巩义等主要仓库

### SMM铝棒库存
- 搜索触达：`SMM 铝棒库存 本周`
- 意义：铝棒库存反映加工端需求，领先于铝锭库存变化

### LME铝库存
- 搜索触达：`LME aluminum inventory latest` / `LME铝库存`
- 意义：海外需求信号，影响内外价差

---

## 三、成本数据源

### 安泰科（Antaike）—— 最权威成本数据
- 搜索触达：`安泰科 电解铝完全成本` / `安泰科 电解铝利润 月度`
- 数据：行业加权平均完全成本（含税）、各省成本分布、行业利润测算
- 发布频率：月度

### 百川盈孚
- 搜索触达：`百川盈孚 电解铝成本` / `百川盈孚 氧化铝价格`
- 数据：细分原料成本追踪（氧化铝、电价、阳极）

### 动力煤价格（影响火电铝成本）
- 搜索触达：`秦皇岛动力煤价格 今日` / `5500大卡动力煤现货`
- 意义：每吨煤价变动约影响火电铝成本35-50元/吨

---

## 四、供给数据源

### 国家统计局
- 搜索触达：`国家统计局 电解铝产量 月度`
- 数据：中国月度电解铝产量，最权威官方数据

### SMM/百川 开工率数据
- 搜索触达：`电解铝开工率 本月` / `云南电解铝产能运行`
- 重点跟踪：云南开工率（水电铝核心产区）

### 云南能源局公告
- 搜索触达：`云南电力调度 铝厂 限电通知`
- 季节性规律：11月-次年5月为枯水期，警惕限电减产

---

## 五、需求数据源

### 汽车行业数据（铝轻量化最大增量）
- 搜索触达：`中国汽车销量 月度` / `新能源汽车产量 最新`
- 发布方：中国汽车工业协会，每月10日前后

### 光伏装机数据
- 搜索触达：`光伏新增装机 月度 国家能源局`
- 发布方：国家能源局，每月中旬

### 家电排产数据
- 搜索触达：`家电排产 产业在线 最新` / `空调排产 月度`
- 发布方：产业在线，月度

## 七、A股标的数据源

### 股价与财务
- 搜索触达：`601600 中国铝业 股价 今日` 
- 或用 `000807 云铝股份 最新股价`
- 年报/季报：通过搜索 `中国铝业 2024年报 归母净利润` 等

### 券商研报
- 搜索触达：`中国铝业 研报 最新` / `铝行业 研报 2025`
- 重点关注：国泰君安、中信建投、广发证券有色金属团队

## 八、数据采集优先级

```
高频（每次分析必查）:
  沪铝价格 → LME铝价 → 氧化铝现货价 → 铝锭社会库存 → 现货升贴水

中频（每周更新）:
  行业开工率 → 铝棒库存 → 动力煤价 → 汽车/光伏数据

低频（每月更新）:
  安泰科完全成本 → 国统局产量 → 上市公司财报 → 券商研报
```
