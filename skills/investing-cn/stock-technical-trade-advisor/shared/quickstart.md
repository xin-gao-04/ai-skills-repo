# 快速上手指南

## 两种使用方式对比

| | 版本A：纯对话（搜索模式） | 版本B：Python+数据（完整模式） |
|--|--------------------------|-------------------------------|
| 需要编程 | ❌ 不需要 | ✅ 需要运行Python |
| 数据精度 | 约60-70%（依赖搜索结果） | 约95%（完整历史数据） |
| 价位精确度 | 参考级（±1-2%误差） | 精确到小数点后两位 |
| 情绪数据 | 有（搜索获取） | 有（akshare实时获取） |
| 适合场景 | 快速初判、随时查询 | 认真分析、实际操作前 |

## 版本A：直接在Claude对话中使用（无需编程）

### 步骤
1. 创建一个 Claude Project，上传以下4个文件：
   - `SKILL.md`
   - `references/indicators.md`
   - `references/risk-management.md`
   - `references/sentiment.md`

2. 在 Project 中发送：
```
对中国铝业（601600）进行完整技术分析，
帮我判断当前是否适合买入，以及具体的入场区间和止损位
```

3. Claude 会自动进入搜索模式，联网采集数据并输出完整报告。

## 版本B：Python脚本 + Claude分析（推荐）

### 步骤一：安装依赖
```bash
pip install akshare pandas numpy
```

### 步骤二：采集数据
```bash
# 分析中国铝业，获取250个交易日数据
python scripts/fetch_stock_data.py --code 601600 --days 250

# 输出文件：601600_ta_data_YYYYMMDD.json
```

### 步骤三：上传并分析
1. 将生成的 JSON 文件上传到 Claude
2. 在配置了本 Skill 的 Project 中发送：
```
数据文件已上传，请对601600中国铝业进行完整技术分析。
我的持仓意图是中波段（2-6周），重点帮我找入场区间和设置止损位。
```

## 常用分析指令模板

### 寻找入场点
```
对[股票名称（代码）]进行技术分析。
目标：寻找买入入场点（趋势初段）。
持仓周期：[短线/波段/中长线]。
请给出入场区间、止损位、第一止盈目标，以及具体的仓位建议。
```

### 判断当前持仓
```
我已持有[股票名称]，成本价[XX元]。
请分析当前技术面状态：
1. 当前处于哪个量价阶段？
2. 趋势是否仍然健康？
3. 止损位应该设在哪里？
4. 第一止盈目标在哪里？
```

### 设置止损止盈
```
对[股票名称（代码）]进行止损止盈分析。
当前价格：[XX元]
已知入场价：[XX元]
请基于技术关键位，给出：
- 严格止损位（必须执行）
- 第一止盈目标
- 第二止盈目标（强趋势延伸）
- 移动止盈规则
```

## 分析频率建议

| 持仓类型 | 建议复盘频率 | 关注重点 |
|---------|------------|---------|
| 短线（1-5天） | 每日收盘后 | 日线MACD、成交量、次日情绪 |
| 波段（2-6周） | 每周一次 | 周线趋势、MA20支撑 |
| 中长线（1-6月）| 每月一次 | 月线趋势、基本面变化 |

## 与铝行业基本面Agent配合使用

技术面和基本面是互补的：
- **基本面**（aluminum-investment-agent）：告诉你为什么值得买，价值几何
- **技术面**（ta-analyst-agent）：告诉你什么时候买，买在哪个价位

**最佳组合流程：**
1. 用 aluminum-investment-agent 判断标的是否值得配置（基本面支撑）
2. 用 ta-analyst-agent 找具体入场时机（技术面择时）
3. 两者信号同向 → 提高仓位
4. 两者信号矛盾 → 以基本面为方向，以技术面为操作时机

## 注意事项

1. **A股特殊性**：A股受政策影响大，纯技术分析有局限性，重大政策窗口期谨慎操作
2. **数据时效**：JSON数据应在分析当日或前一交易日采集，超过3天需重新采集
3. **止损必须执行**：技术分析给出的止损位是客观信号，不执行止损是最常见的亏损原因
4. **仓位控制优先于入场点**：即使找到了好的入场点，超额仓位仍然是风险
5. **本工具仅供辅助决策**：市场永远有不确定性，任何分析都不能保证盈利
