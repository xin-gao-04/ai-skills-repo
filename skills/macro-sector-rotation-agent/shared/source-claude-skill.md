# 宏观板块轮动分析 Agent

## 工作流概述

本 Agent 分为 **5个串行阶段**，每阶段输出结构化结果，最终汇总为可执行的板块配置建议。

```
阶段1: 宏观与市场数据采集
    ↓
阶段2: 宏观状态与流动性判断
    ↓
阶段3: 板块轮动信号识别
    ↓
阶段4: 板块打分与排序
    ↓
阶段5: 生成板块策略报告
```

详细数据源见 `references/data-sources.md`
轮动信号模型见 `references/rotation-signal-model.md`
板块评分框架见 `references/sector-scoring-model.md`
宏观映射手册见 `references/regime-sector-playbook.md`
输出质检清单见 `references/quality-checklist.md`

## 阶段1：宏观与市场数据采集

**目标：** 获取影响A股板块轮动的核心变量，形成 `macro_snapshot`。

### 1.1 宏观与政策变量（优先级最高）

依次搜索并记录：

| 数据项 | 搜索词 | 期望获取 |
|---|---|---|
| 中国制造业PMI | `中国制造业PMI 最新` | 最新值、前值、荣枯线位置 |
| 社融/信贷脉冲 | `中国 社融 存量增速 最新` | 同比、边际变化 |
| CPI/PPI | `中国 CPI PPI 最新` | 通胀与通缩方向 |
| 10Y国债收益率 | `中国10年期国债收益率 今日` | 利率方向 |
| 美联储政策路径 | `Federal Reserve latest statement rate path` | 鹰/鸽倾向 |
| 美债10Y | `US 10Y Treasury yield today` | 全球流动性约束 |
| 美元指数DXY | `DXY US dollar index today` | 风险偏好环境 |
| 核心政策事件 | `国务院 产业政策 最新` | 财政/产业/监管导向 |

### 1.2 风险偏好与资金变量

| 数据项 | 搜索词 | 期望获取 |
|---|---|---|
| 北向资金净流入 | `北向资金 今日 净流入` | 单日/周累计 |
| 两融余额 | `融资融券余额 最新` | 风险偏好强弱 |
| 全A成交额 | `A股 成交额 今日` | 量能环境 |
| 风格指数强弱 | `沪深300 中证1000 创业板 指数 今日` | 大小盘风格切换 |

### 1.3 板块与行业价格表现

覆盖申万一级（或中信一级）主要板块，至少包括：
金融、周期（有色/煤炭/钢铁/化工）、消费、科技、医药、公用事业、军工、地产链。

输出：
- 近5日涨跌幅
- 近20日涨跌幅
- 相对沪深300超额收益
- 成交额与换手变化

### 1.4 国际局势与大宗商品变量

| 数据项 | 搜索词 | 用途 |
|---|---|---|
| 原油价格 | `Brent crude oil price today` | 通胀/能源链 |
| 黄金价格 | `gold price today` | 避险强弱 |
| 铜价 | `LME copper price today` | 全球增长代理 |
| 地缘冲突/关税事件 | `latest geopolitical tensions tariffs` | 风险与结构性机会 |

**采集完成后，将结果整理为 `macro_snapshot`，进入阶段2。**

### 1.5 缺失与时效处理（必须执行）

- 高频市场数据（指数、成交额、北向）：若当日不可得，使用最近1个交易日数据，并注明日期
- 月度宏观数据（PMI、社融、CPI/PPI）：若当月未发布，使用上月数据并标注“待更新”
- 事件类信息（政策/地缘）：至少交叉验证2个来源，若冲突则保留分歧并降低结论置信度
- 若超过30%关键变量缺失，自动降级为“观察版报告”，不得给出强超配结论

## 阶段2：宏观状态与流动性判断

**目标：** 判断当前市场所处宏观环境，得到 `macro_regime`。

### 2.1 宏观四象限框架

基于“增长 + 通胀”给出状态：

- 复苏：增长上行，通胀温和
- 过热：增长上行，通胀抬升
- 滞胀：增长下行，通胀上行
- 衰退：增长下行，通胀下行

### 2.2 流动性状态

结合中美利率、美元、社融脉冲、市场成交额，判断：

- 宽松 / 中性 / 收紧
- 增量资金偏好：大盘价值 / 小盘成长 / 红利防御

### 2.3 政策与事件冲击

从阶段1提取政策与国际事件，给出：

- 政策主线（稳增长、科技自主、出海、能源安全等）
- 事件冲击方向（利多、利空、中性）
- 持续性评估（短期交易 / 中期趋势）

**输出格式：**

```
宏观状态: 复苏/过热/滞胀/衰退
流动性状态: 宽松/中性/收紧
政策主线: [2-3条]
国际风险等级: 低/中/高
市场主风格: 价值/成长/防御/均衡
```

并读取 `references/regime-sector-playbook.md`，输出“基准配置方向”。

## 阶段3：板块轮动信号识别

**目标：** 用统一信号框架识别下一阶段可能占优板块。

读取 `references/rotation-signal-model.md`，完成以下信号打分：

### 3.1 趋势动量信号
- 板块近20日、60日相对强弱
- 是否突破关键均线（20/60日）
- 量价配合情况

### 3.2 估值与拥挤度信号
- 当前估值分位（PE/PB/PS）
- 近3个月涨幅与换手拥挤度
- 与盈利预期变化是否匹配

### 3.3 基本面与盈利修正信号
- 行业景气指标改善/走弱
- 分析师盈利预期上修/下修
- 订单、库存、价格链条验证

### 3.4 政策与地缘催化信号
- 政策确定性强度
- 事件是否具备可持续催化
- 受益板块是否明确且可交易

将每个板块形成 `signal_scorecard`，并标注：
- 主驱动因子
- 反向风险因子
- 信号时效（1-2周 / 1-3个月）
- 证据完整度（高/中/低）

## 阶段4：板块打分与排序

**目标：** 基于统一权重得到板块配置优先级。

读取 `references/sector-scoring-model.md`，按维度打分：

| 维度 | 权重 | 说明 |
|---|---|---|
| 宏观匹配度 | 25% | 与 `macro_regime` 适配度 |
| 轮动强度 | 25% | 趋势动量与相对强弱 |
| 基本面改善 | 20% | 盈利与景气边际变化 |
| 估值性价比 | 15% | 分位与盈利匹配 |
| 政策/事件催化 | 15% | 催化确定性与持续性 |

### 4.1 评分输出要求

输出 Top 5 板块与 Bottom 3 板块，格式：

```
板块: 半导体
综合得分: 78/100
当前评级: 超配
主逻辑: 政策支持+盈利上修+相对强势
关键风险: 估值拥挤，若成交额回落则回撤加大
观察周期: 2-6周
```

### 4.2 板块→标的映射（可选）

如用户要求，给出每个优先板块的2-3只代表性A股，
并说明“板块逻辑到个股逻辑”的传导条件。

### 4.3 冲突信号裁决（必须输出）

当以下情况出现时，必须明确裁决：
- 宏观映射看多，但动量转弱
- 动量强势，但估值与拥挤度极高
- 政策催化强，但基本面未验证

裁决格式：
```
冲突类型:
当前采用:
理由:
切换阈值:
```

**目标：** 汇总前4阶段，输出结构化策略报告。

### 5.1 报告结构（ALWAYS 使用此模板）

```markdown
# A股宏观板块轮动周报
**报告日期：** YYYY-MM-DD
**报告类型：** 定期跟踪 / 事件驱动

---

## 一、核心结论（TL;DR）
[3-5条可执行结论：当前主线、回避方向、风格建议]

## 二、宏观与流动性环境
[宏观四象限、流动性状态、政策主线、国际风险]

## 三、轮动信号看板
[板块信号总表：动量、估值、基本面、催化]

## 四、板块配置建议
[超配/标配/低配 清单与理由]

## 五、重点板块深度解读
[Top 3板块逐一说明驱动、验证指标、失效条件]

## 六、风险与对冲思路
[3-5个主要风险 + 对应防御板块/风格建议]

## 七、下期观察清单
[下周必须跟踪的数据与事件日历]

## 八、置信度与证据覆盖

### 5.2 配置建议格式

每个板块建议必须包含：

```
板块：

时间维度：短期（1-4周）/ 中期（1-3个月）
核心逻辑：
触发条件：
失效条件：

### 5.3 文件输出规则

报告保存为 Markdown，文件名格式：
`macro-sector-weekly-YYYY-MM-DD.md`

若同日多次运行，可使用：
`macro-sector-weekly-YYYY-MM-DD-HHMMSS.md`

同时在对话中给出不超过300字摘要。

输出前必须读取 `references/quality-checklist.md` 并逐条自检。

## 执行注意事项

1. **时效性优先：** 所有关键数据标注日期
2. **缺失值处理：** 使用最近一期数据并标注替代来源
3. **观点可证伪：** 每个结论附触发条件与失效条件
4. **中文输出：** 全文中文，保留必要英文缩写（PMI、DXY、FOMC）
5. **风险声明：** 报告末尾必须保留投资风险提示
6. **结论分级：**
   - 置信度高：可给出超配/低配
   - 置信度中：以标配和观察为主
   - 置信度低：只给跟踪清单，不给明确仓位建议

## 场景化执行模式

根据用户需求自动匹配模式：

1. **完整模式（默认）**：执行全部5阶段，输出完整周报
2. **快照模式（盘中/临时）**：只执行阶段1-3，输出“当前主线 + 风险”
3. **事件模式（突发政策/地缘）**：先输出24小时临时判断，再补全完整流程
4. **离线模式（无法联网）**：仅基于用户提供数据与历史框架输出“假设性分析”，明确限制
