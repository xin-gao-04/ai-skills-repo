# 利润弹性模型 - 各标的基础参数

## 核心公式

### 电解铝吨利润（行业通用）
```
吨利润（税前）= 铝现货价（不含税）- 综合完全成本（不含税）

铝现货价（不含税）= 铝锭含税价 ÷ 1.13
综合完全成本（不含税）= 完全成本（含税）÷ 1.13

完全成本（含税）≈ 氧化铝成本 + 电力成本 + 阳极成本 + 其他制造成本 + 期间费用
                = 氧化铝价×1.93 + 电价(元/度)×13500 + 阳极价×0.47 + 其他

阳极价（预焙阳极）约 3500-4500 元/吨，变化不大
其他（人工、折旧、期间费用等）约 1500-2000 元/吨
```

### 氧化铝吨利润
```
氧化铝吨利润 = 氧化铝现货价 - 氧化铝完全成本
氧化铝完全成本 ≈ 铝土矿成本 + 烧碱成本 + 能源成本 + 其他
                ≈ 铝土矿价×2.5 + 烧碱价×0.08 + 能源 + 约500
（参考值：2025年氧化铝行业加权完全成本约2800-3000元/吨）
```

## 各标的基础参数档案

### 中国铝业（601600）

**产能规模（2024年报数据）：**
- 氧化铝产能：约2200万吨/年（全球第一）
- 电解铝产能：约760万吨/年（国内第一）
- 2024年实际产量：氧化铝1687万吨、电解铝761万吨

**成本结构：**
- 电解铝制造成本：约15,500元/吨（2024年，不含税）
- 自备电比例：约18%（2024年下降，主要依赖网电）
- 氧化铝自给率：约55%（内部转移定价，外购部分按市价）
- 铝土矿自给率：约62%

**收入结构：**
- 电解铝板块：主要收入来源，外售约760万吨
- 氧化铝外售：约635万吨（超出自用部分），是重要利润贡献
- 其他：煤炭、新能源、精细氧化铝等

**利润弹性参数：**
```
铝价弹性：
  电解铝外售量 ≈ 760万吨
  铝价+100元/吨 → 税前增量 = 760万吨 × 100 = 7.6亿元
  归母净利增量 ≈ 7.6 × (1-25%所得税) × (1-35%少数股东) ≈ 3.7亿元
  即：铝价每变动100元/吨，归母净利变动约±3.7亿元

氧化铝价格净弹性（综合外售收益与内购成本）：
  外售635万吨：+100元/吨 → 税前增量+6.35亿
  外购氧化铝（电解铝产量×1.93×外购比例45%）≈ 660万吨
  外购成本增加：660万吨 × 100 = 6.6亿（成本上升）
  净效果：-0.25亿/100元（轻微负向，中铝是氧化铝净买入方）
  
电价弹性：
  综合用电量 = 761万吨 × 13500度 = 1027亿度
  自备电比例18%，网电82%
  网电价格+0.01元/度 → 成本增加 = 1027亿度×82%×0.01 ≈ 8.4亿元/年
  即：综合电价每变动0.01元/度，归母净利变动约±4亿元
```

**关键风险：**
- 网电比例高（82%），枯水季电价上涨影响大
- 国企机制，成本下降效率低于民营竞争对手
- 几内亚铝土矿政策风险

### 云铝股份（000807）

**产能规模：**
- 电解铝产能：约400万吨/年
- 2024年产量：约400万吨（水电铝核心标的）

**成本结构：**
- 电解铝完全成本：约13,000-14,000元/吨（行业最低之一）
- 用电结构：以云南水电为主（70%+），网电比例低
- 无自有氧化铝，全部外购

**利润弹性参数：**
```
铝价+100元/吨 → 归母净利增量 ≈ 400万吨×100×75%×55% ≈ 1.65亿元
  （少数股东权益占比约45%）

氧化铝价格+100元/吨 → 成本增加 = 400万吨×1.93×100 = 7.72亿
  → 归母净利减少约 7.72×75%×55% ≈ 3.2亿元（负向弹性大）

电价弹性：与水电调度挂钩，枯水期减产>电价上涨，主要是量的风险而非价的风险
```

**投资特点：**
- 铝价弹性最大，涨价受益最明显
- 氧化铝上涨是最大风险（全部外购）
- 枯水期（11月-次年5月）产量约下降15-25%，但成本也随之降低
- 水电铝的"绿色"属性对抗CBAM碳关税

### 神火股份（000933）

**产能规模：**
- 电解铝产能：约160万吨/年（新疆为主）
- 煤炭产能：约1300万吨/年（重要）

**成本结构：**
- 电解铝完全成本：约14,000-15,000元/吨
- 煤电铝一体化，自备电比例高（约70%）
- 无自有氧化铝，全部外购

**利润弹性参数：**
```
铝价+100元/吨 → 归母净利增量 ≈ 约0.7亿元
氧化铝+100元/吨 → 归母净利减少约 -1.3亿元

煤炭业务对冲逻辑：
  铝价下跌通常对应经济走弱 → 煤价也可能下跌（成本降低）
  铝价上涨通常对应经济强劲 → 煤炭收入也可能增长
  两者有一定自然对冲效果，使业绩波动小于纯铝企业
```

**投资特点：**
- 煤铝双主业，业绩稳定性好于纯铝企业
- 铝价弹性低于云铝
- 新疆低成本煤电，成本优势明显
- 适合稳健型持仓

---

### 天山铝业（002532）

**产能规模：**
- 电解铝产能：约200万吨/年（新疆）
- 电力自给率高

**成本结构：**
- 完全成本：约13,500-14,500元/吨（行业低成本）
- 煤电一体化，成本控制强

**利润弹性参数：**
```
铝价+100元/吨 → 归母净利增量 ≈ 约0.85亿元
历史PE弹性大，价格上涨时股价放大效应强
```

**投资特点：**
- 成本低、弹性大
- 体量较小，流动性风险
- 历史上在铝价上行周期股价弹性最大

| 中性：20000 | 3000 | 约100亿 | 约40亿 | 约35亿 |
| 当前：23000 | 2800 | 约170亿+ | 约80亿 | 约45亿 |

## 重要提示

1. 以上参数来源于2024年报和公开研究，需在每季度报告后更新
2. 产量数据受云南水电调度、设备检修等影响，实际产量会有偏差
3. 少数股东权益比例各公司不同，影响归母净利计算，以最新年报为准
4. 套保比例：各公司的套保策略不同，套保会平滑价格弹性，需关注季报披露
